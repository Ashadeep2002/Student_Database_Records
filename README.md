# 🎓 Student Database Management System (SQL)

## 📖 Overview

This project is a relational database system built using SQL to manage student records, teachers, courses, enrollments, and grades. It demonstrates database design, table relationships, joins, aggregate functions, and CRUD operations.

---

## 🗄️ Database Structure

The database contains the following tables:

Table Name	- Description
Students	  - Stores student information
Teachers	  - Stores teacher information
Courses	    - Stores course details
Enrollments	- Tracks student course enrollments
Grades	    - Stores student grades

---

## 🛠️ Features

Create database and tables
Insert records
Update records
Delete records
Use Primary Keys and Foreign Keys
Perform JOIN queries
Aggregate functions (COUNT, AVG, MAX)
Course enrollment tracking
Grade management system

DDL (CREATE DATABASE, CREATE TABLE)
